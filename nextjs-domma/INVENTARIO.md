## 🚀 COMO USAR

### 1. EXTRAIR
```bash
unzip domma-juridico-nextjs.zip
cd nextjs-domma
```

### 2. INSTALAR
```bash
npm install
```

### 3. RODAR
```bash
npm run dev
```

### 4. ACESSAR
http://localhost:3000
