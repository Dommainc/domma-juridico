# 🚀 GUIA COMPLETO - DOMMA JURÍDICO NEXT.JS

## 📋 PRÉ-REQUISITOS

1. **Node.js 18+** - https://nodejs.org/
2. **Git** (opcional)
3. **VS Code** (recomendado)

---

## 📦 INSTALAÇÃO PASSO A PASSO

### 1. CRIAR PASTA DO PROJETO

```bash
mkdir domma-juridico
cd domma-juridico
```

### 2. COPIAR TODOS OS ARQUIVOS

Copie TODOS os arquivos que criei para dentro da pasta `domma-juridico/`

Estrutura deve ficar assim:
```
domma-juridico/
├── app/
├── components/
├── lib/
├── public/
├── .env.local
├── package.json
├── next.config.js
├── tailwind.config.ts
└── tsconfig.json
```

### 3. INSTALAR DEPENDÊNCIAS

```bash
npm install
```

Aguarde a instalação (pode demorar 2-3 minutos).

### 4. RODAR O PROJETO

```bash
npm run dev
```

Abra o navegador em: **http://localhost:3000**
